Rules

  • To participate in the quiz, players must provide a valid email address for logging in.
  • Each player starts with 3 lives.
  • Each correct answer earns the player 100 points.
  • Incorrect answers do not result in a point deduction but will cost the player one life.
  • The quiz will cover events from the English Premier League that occurred in the last five years.
  • Questions will be presented in multiple-choice format.
  • Players must select the correct answer from the given four options.
  • Players will be presented with a series of questions related to English Premier League events.
  • If you answer incorrectly, you will lose one life, but the game will continue with the next question.
  • You can skip a question if you are unsure, but no points will be awarded or deducted for skipped questions.
  • The game ends when you lose all 3 lives or you have completed all the questions in the quiz.
  • Cheating or using external resources to answer questions is not allowed.
  • Fair play and honest participation are expected from all players.
